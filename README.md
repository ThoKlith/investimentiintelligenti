# Investimenti Intelligenti

Sito in Next.js (App Router) con UI/UX rinnovata e contenuti placeholder.

## Comandi utili

Avvio in sviluppo:

```bash
npm run dev
```

Build produzione:

```bash
npm run build
```

Lint:

```bash
npm run lint
```
